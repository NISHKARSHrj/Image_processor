# Image Processor 🚀

Image Processor is a modern batch image processing web application built with Flask, HTML, CSS, and Vanilla JavaScript.

🌐 Live Demo:
https://imageprocessor-d9z7.onrender.com

--------------------------------------------------
✨ FEATURES
--------------------------------------------------

• Drag & Drop Image Upload
• Live Image Preview Grid
• Batch Image Processing
• Sharpen Filter
• Grayscale Conversion
• Image Rotation
• Export as PNG / JPEG / WEBP
• Real-Time Processing Logs
• Responsive & Minimal UI

--------------------------------------------------
🛠️ TECH STACK
--------------------------------------------------

Frontend:
- HTML5
- CSS3
- Vanilla JavaScript

Backend:
- Python
- Flask
- Pillow (PIL)

⚙️ INSTALLATION
--------------------------------------------------

1. Clone Repository

git clone https://github.com/your-username/image-processor.git
cd image-processor

2. Create Virtual Environment

python -m venv venv

Activate Environment:

Windows:
venv\Scripts\activate

Linux/macOS:
source venv/bin/activate

3. Install Dependencies

pip install -r requirements.txt

4. Run Application

python app.py

--------------------------------------------------
🧠 HOW IT WORKS
--------------------------------------------------

Frontend:
- Handles image uploads
- Displays image previews
- Controls processing settings
- Shows progress logs

Backend:
- Receives uploaded images
- Processes images using Pillow
- Creates ZIP archive
- Returns processed files
